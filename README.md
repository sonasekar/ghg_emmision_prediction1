## 📌 Project Title

**Project 2 (P2): Greenhouse Gas Emission Prediction**

---

## 🔍 Objective

To leverage Artificial Intelligence techniques to **analyze and predict greenhouse gas (GHG) emissions** using industry-level datasets, thereby contributing toward sustainable development goals and climate change mitigation.

---

## 🎯 Problem Statement

Industries across the globe contribute significantly to GHG emissions. However, many lack the tools to effectively **predict future emissions** based on their operations and outputs. This project aims to solve this problem using **machine learning models** trained on historical emission data.

---

## 💡 Proposed Solution

Build a predictive system that:
- Analyzes past GHG emission data
- Identifies key influencing factors
- Predicts future emission levels
- Helps industries make eco-friendly decisions

---

## 🛠️ Tech Stack

| Layer         | Technology               |

| Data Handling | Pandas, NumPy            |
| Visualization | Matplotlib, Seaborn      |
| Modeling      | Scikit-learn, RandomForest |
| Backend       | Python                   |
| Notebook      | Jupyter Lab / Google Colab |

---

## 🔁 Workflow

1. 📥 Load the supply chain emission dataset
2. 🧼 Clean and preprocess the data
3. 📊 Explore and visualize patterns
4. 🧠 Build and train regression models
5. 📈 Evaluate performance (RMSE, R²)
6. 💾 Save and export the trained model

---

## ✅ Features

- Supports historical GHG analysis from multiple industries
- Predicts future CO₂ emission levels
- Easy-to-visualize output and insights
- Saves models for future reuse or deployment
